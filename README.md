# Hackathon-II: Evolution of Todo App

A multi-phase project demonstrating the evolution of a Todo application from a simple console app to a full-stack production-ready application with AI capabilities.

## Project Structure

```
Hackathon-II-Evolution_Of_Todo_App/
├── phase-1-console/          # Phase 1: Python Console Application
├── phase-2-fullstack/        # Phase 2: Full Stack Web Application
├── phase-3_ai_chatboat/      # Phase 3: AI Chatbot Integration (Coming Soon)
├── phase-4-production/       # Phase 4: Production Deployment (Coming Soon)
└── README.md
```

## Phases

### Phase 1: Console Application
A Python-based command-line Todo application with CRUD operations.
- **Status:** Completed
- **Technologies:** Python, JSON storage
- **Location:** `phase-1-console/`

### Phase 2: Full Stack Web Application
A modern web application with separate frontend and backend.
- **Status:** Completed
- **Technologies:**
  - Backend: FastAPI, SQLModel, SQLite, JWT Authentication
  - Frontend: Next.js 16+, TypeScript, Tailwind CSS
- **Location:** `phase-2-fullstack/`

### Phase 3: AI Chatbot Integration
AI-powered chatbot for natural language task management.
- **Status:** Coming Soon
- **Location:** `phase-3_ai_chatboat/`

### Phase 4: Production Deployment
Production-ready deployment with DevOps best practices.
- **Status:** Coming Soon
- **Location:** `phase-4-production/`
