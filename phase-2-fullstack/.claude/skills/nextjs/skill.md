# NextJS Skill

## Purpose
This skill provides expertise in NextJS 14+ development with App Router, Server Components, and modern React patterns.

## Capabilities
- Create NextJS pages and layouts using the App Router
- Implement Server and Client Components appropriately
- Handle data fetching with React Server Functions
- Manage state using React Hooks and Context
- Implement forms and handle submissions
- Work with NextJS routing and navigation
- Apply Tailwind CSS for styling
- Implement API routes when needed
- Handle authentication flows
- Optimize for performance and SEO

## Best Practices
- Use Server Components by default unless client interactivity is needed
- Leverage React Server Functions for data fetching
- Follow NextJS file-based routing conventions
- Use TypeScript for type safety
- Implement proper error boundaries
- Apply loading and suspense patterns appropriately
- Follow accessibility guidelines
- Use NextJS image optimization features
- Implement proper meta tags for SEO

## File Structure
- Place page components in the app directory following NextJS conventions
- Use the lib directory for shared utilities
- Place reusable components in the components directory
- Store API client code in the lib directory
- Use the public directory for static assets
